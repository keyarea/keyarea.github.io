方法递归调用（了解）

方法的递归调用指的是一个方法自己调用自己的形式。

递归：

  • 方法必须有一个递归的结束条件
  • 方法在每次递归处理的时候一定要做出一些变更

范例：实现1-100的累加操作

【调用】第一次执行sum（）方法，主方法执行。return 100 + sum（100-1）；
【递归】第二次执行sum（）方法：return 99 + sum（99-1）；
【递归】第二次执行sum（）方法：return 98+ sum（98-1）；
…..
