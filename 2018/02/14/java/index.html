定义一个学生选课的操作表，三张数据表：

  • 学生表：学生编号，姓名，年龄

  • 学生-课程关系表：学生编号，课程编号，成绩

要求：可以实现以下的信息输出：

  • 可以找到一门课程，以及参加此课程的所有学生信息和她的成绩。
