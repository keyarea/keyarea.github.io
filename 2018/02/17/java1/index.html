Java 综合案例：数组操作

发表于 2018-02-17 | 分类于 Java
字数统计: 1.8k | 阅读时长 ≈ 8

现在要求定义一个数组（整型数据）的操作类，这个类有如下几点要求：

  • 数组的大小由类创建对象的时候动态决定；
  • 可以通过类向数组中进行数据的保存，保存的时候需要考虑到数组的大小问题；
  • 如果发现数组空间不足，则可以进行数组长度的动态扩充；
  • 可以取得数组的全部数据（增加顺序）；

而后继续在这个基础上要求扩展两个派生类：

  • 可以进行数组的排序操作；
  • 可以进行数组的反转处理；
