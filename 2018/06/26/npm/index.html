NPM 国内慢的问题解决

发表于 2018-06-26 | 分类于 Node.js
字数统计: 506 | 阅读时长 ≈ 2

NPM 是随同 NodeJS 一起安装的包管理工具，我们经常使用它来下载第三方包到本地。
但在使用 NPM 过程很多人估计都知道，在国内下载第三方包的速度极其之慢。
下面是我整理的几种比较好用的解决办法。

#　淘宝NPM镜像(cnpm)

它是一个完整 npmjs.org 镜像，你可以用此代替官方版本(只读)，同步频率目前为 10分钟 一次以保证尽量与官方服务同步。
淘宝 NPM 镜像地址：https://npm.taobao.org
